## About me

I'm a front-end developer with a growing interest in the full product experience - from thoughtful interfaces to the backend functionality that supports them.

Since late 2025, I've been completing an intensive front-end development program at **Lexicon**. The program combines full-time studies with hands-on project work and an internship, giving me experience turning ideas and requirements into working digital products.

## Experience

### Blueye - Front-end development intern

During an internship of roughly two to three months, I:

- helped design and build modern websites while developing both my front-end and backend experience;
- contributed to mobile app development by cleaning up and refining the interface;
- added in-app messaging based on the existing web experience; and
- implemented message notifications for the mobile app.

## Education

### Lexicon - Front-end Developer

Lexicon's 100-day, full-time front-end curriculum focuses on building responsive and accessible web applications. My studies have included:

- semantic HTML, CSS Grid, Flexbox, responsive design, and accessibility fundamentals based on WCAG;
- TypeScript, DOM manipulation, asynchronous programming, and the Fetch API;
- React and Next.js with JSX/TSX, components, props, routing, state management, and API integration; and
- Git, GitHub Projects, and collaborative development using Scrum.
